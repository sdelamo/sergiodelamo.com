## Install NPM

Install `npm`.  You can [install in MacOS with homebrew](https://www.npmjs.com/package/homebrew). 

## Install AWS CDK CLI
[Install AWS CDK CLI](https://docs.aws.amazon.com/cdk/v2/guide/cli.html)

```
npm install -g aws-cdk  
``
